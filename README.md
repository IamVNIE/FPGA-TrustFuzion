# FPGA-TrustFuzion

### Requirements

      1. Xilinx Impact (Any Version)
      2. USB-UART Driver for your FPGA
      3. Windows OS

### Setup

      Clone or Download folder. 
      Open download.cmd and change the path to Impact (ONLY if you have not installed Xilinx in default location)
      
### IDPVMS Execution

      Follow the tutorial
      
### Post Processing
